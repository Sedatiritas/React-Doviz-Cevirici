# React Döviz Çevirici
Bu proje, **React** kullanılarak geliştirilmiş basit bir döviz çevirici uygulamasıdır.  
Kullanıcıların bir para birimini başka bir para birimine çevirmesine olanak sağlar.
## Özellikler
- Anlık döviz kuru hesaplama
- Kullanıcı arayüzü ile kolay kullanım
- React state ile dinamik güncelleme
## Teknolojiler
- React
- JavaScript
- CSS
## Kurulum
```bash
git clone https://github.com/Sedatiritas/React-Doviz-Cevirici
cd React-Doviz-Cevirici
npm install
npm start
